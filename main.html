ยินดีต้อนรับเข้าสู่เว็ปไซต์เพื่อการศึกษาสำหรับการเรียนการสอน วิชาพื้นฐานการสร้างเว็ปไซต์

คำอธิบายรายวิชา ศึกษาและปฎิบัติเกี่ยวกับโครงสร้างขอองเว็ปไซต์ หลักการออกแบบเว็ปไซต์ด้วยโปรเเกรม HTML5 การตกเเต่ง การแทรกรูปภาพหรือสื่อสิ่งต่างๆ การสร้างตาราง การสร้าง การเชื่อมโยงในแบบต่างๆ การส้รางเฟรม การออกแบบและสร้างฟอร์ม การแทรก Java script การทดสอบการทำงานของเว็ปไซต์ การอัพโหลดเว็ปไซต์ไปยังเว็ปเซิร์ฟเวอร์
